# Unicorn template Markup

It is my version of markup of of [this template](http://graphicburger.com/unicorn-psd-template/).

In this project I did use:
- Gulp;
- BrowserSync;
- Sass
- Bootstrap

For use it:
1. Download/Clone repository;
2. Enter in folder from terminal;
3. Execute command:
    ```javascript 
    npm install
    ````
4. And then:
    ```javascript 
    gulp
    ````
